# logos

A CLI tool for scaffolding research projects with a structured agenda.

## Install

```bash
git clone https://github.com/YOUR_USERNAME/logos.git
cd logos
sudo ./install.sh
```

Or install to a custom location:

```bash
INSTALL_DIR=~/.local/bin ./install.sh
```

## Usage

```bash
logos myproject
```

This creates a `myproject/` directory with:
- A git repo initialized
- `agenda.md` — a research agenda template

## Uninstall

```bash
rm /usr/local/bin/logos
```
